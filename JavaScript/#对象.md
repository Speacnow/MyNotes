#函数
1. 什么是函数？  
	实现是特定功能的n条语句的封装体。
	可执行 其他类型数据不可执行
	
2. 为什么要用函数？
	提高代码复用
	便于阅读交流
	
3. 如何定义函数？
	
	函数声明：
  
  ```javascript
    function fn(){}
  ```
  
  表达式:
  
    ```js
    var fn =function(){}
    ```
  
4. 如何调用函数？
  fn() 直接调用
  obj.fn() 通过对象调用
  new fn() new调用
  fn.call / apply()   临时让fn成为obj的方法进行调用
